Criando um repositório:

Dentro da pasta que deseja versionar ou monitorar, clicar com o botão direito do mouse e escolher a opção :
"Git Bash Here". Isso fará com que abra o terminal.

Primeiro deve-se informar ao git um usuário(o mesmo utilizado no github, gitlab etc...).
Inserindo os comandos:
git config --local user.name"nome do usuário"
git config --local user.email"endereço de email"

Observação: no final de cada comando, deve-se digitar a tecla(enter)

Comandos Git

Para solicitar a criação do repositório local, digite no terminal:
git init
Exibirá a mensagem: "Initialized empty Git repository in ..."

Mostrar o status:
git status

Adicionar arquivos para versionamento/monitoramento para serem commitados:
git add
git add"nome do arquivo"
Adiciona todos os arquivos para serem commitados
git add .

Salvar uma versão ou um estado do código:
git commit
Salvar uma versão ou um estado do código junto com uma breve descrição do que foi feito:
git commit -m"mensagem"

Comando para limpar o terminal: clear
