Criando um repositório:

Dentro da pasta que deseja versionar ou monitorar, clicar com o botão direito do mouse e escolher a opção :
"Git Bash Here". Isso fará com que abra o terminal.

Primeiro deve-se informar ao git um usuário(o mesmo utilizado no github, gitlab etc...).
Inserindo os comandos:
git config --local user.name"nome do usuário"
git config --local user.email"endereço de email"

Observação: no final de cada comando, deve-se digitar a tecla(enter)

Comandos Git

Para solicitar a criação do repositório local, digite no terminal:
git init
Exibirá a mensagem: "Initialized empty Git repository in ..."

Mostrar o status:
git status

Adicionar arquivos para versionamento/monitoramento para serem commitados:
git add
git add"nome do arquivo"
Adiciona todos os arquivos para serem commitados
git add .

Salvar uma versão ou um estado do código:
git commit
Salvar uma versão ou um estado do código junto com uma breve descrição do que foi feito:
git commit -m"mensagem"

Comando para limpar o terminal: clear

Comandos aprendidos:

git init: Inicia um repositório local.

git status: Analiza o repositório local.

git add, git add"nome do arquivo", git add .: Adiciona arquivos para serem commitados.

git commit -m"mensagem": Salva a versão.

git log: Mostra o histórico das versões.

Gerenciar nossas versões com Github + novos comandos.

Criar um repositório no github

Criar um novo repositório no github

Digitar o comando:
git remote add origin https://github.com/FabiMacedo/Git-Recode.git

Em seguida o comando:
git push -u origin master

E assim o repositório local sobe para o github.

E se for feita alguma alteração, é só repetir os comandos:
git add .
git commit -m"mensagem"
git push -u origin master

Apresentamos mais dois novos comandos primordiais para hospedar qualquer projeto onde será necessário o controle de versões com GIT, que são:

git remote add origin url do seu repositório no git
Conecta o repositório local ao nosso repositório no GitHub.

git push origin master
Envia os arquivos e as novas versões para o GitHub.

Para clonar um repositório do github digite o comando:
git clone link do repositório
